Tetbot (Tetris Bot)

Python

A bot that can play Tetris forever

Axie Pairing Tool

Python Javascript Jupyter

A tool to compare genes, and find optimal pairings for large Axie farms

- hexidecimal genes are decoded into readable genes in Javascript
- Python script to calculate compatibility score, and rank all potential partners
